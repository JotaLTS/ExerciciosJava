### ⚙ Funcionalidades

### 👤 Classe Abstrata `Funcionario`
- Realizar login (recebe usuário e senha)  
- Realizar logoff  
- Alterar dados  
- Alterar senha  
- Consultar vendas (método abstrato)

---

### 👔 Classe `Gerente`
- Gerar relatório financeiro  
- Consultar vendas  

---

### 🛒 Classe `Vendedor`
- Realizar venda (incrementa o número de vendas quando acionado)  
- Consultar vendas  

---

### 💼 Classe `Atendente`
- Receber pagamentos (recebe um valor que deve ser incrementado no valor em caixa)  
- Fechar o caixa  
- Consultar vendas
